COVID-19 Dashboard

This project is based on a COVID-19 React web app tutorial made by Colby Fayock. This app is a dynamic COVID-19 dashboard that displays various statistics and information regarding COVID-19 cases across the world using tables, charts and maps.

Map functionality and overall site interactivity can be accredited to the original tutorial, which utilized Gatsby and (React)Leaflet. MaterialUI was used for the tables, as well as ChartJS for the graphs/charts.

Who Would Win?

This project was made in collabration with 3 other classmates for our first class project in Front-End web development. Who Would Win is a fun online application where you can vote on the most epic, weird, or hysterical combinations of characters. We utilized Next.js (a React framework) as the primary foundation of our product.

This allowed us to structure our project into components and pages, each having their own JSX markup (HTML-like syntax used in React projects). From there, each subsequent component and page had their own corresponding styling. Scripting was done through JavaScript, which was embedded within our components. We utilized Google Firestore for our database functionality, which features two collections.
